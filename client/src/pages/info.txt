주요 페이지가 들어가있어야 하는 폴더

제미나이가 추천한 파일 구성

1. LoginPage.jsx (로그인 페이지)

흐름도 위치: 상단 구글 아이디 로그인 구역
역할: 앱의 시작점입니다. 구글 로그인 버튼이 위치하며, 로그인 시 '세과영 이메일(@sasa.hs.kr)'인지 판별하여 메인 페이지로 보낼지, 튕겨낼지 결정하는 로직이 들어갑니다.

2. MainPage.jsx (메인 페이지)

흐름도 위치: 중앙 메인 페이지 구역
역할: 로그인 직후 보게 되는 홈 화면입니다. 흐름도에 나와 있듯 교실 찾기, 검색, 마이페이지로 이동할 수 있는 내비게이션 허브 역할을 합니다.

3. MapPage.jsx (교실찾기 / 지도 페이지)

흐름도 위치: 좌측 하단 교실찾기 페이지 구역
역할: 프로젝트의 핵심 기능입니다.

포함될 UI/기능:

층수 변경 버튼
지도 드래그/줌인/줌아웃
교실 상태(사용 허가, 사용중, 비어있음)에 따른 색상 표시
'비어있는 교실 목록 보여주기' 버튼 (리스트 뷰 전환 기능)

4. SearchPage.jsx (검색 결과 페이지)

흐름도 위치: 중앙 하단 검색 페이지 구역
역할: 메인 페이지나 메뉴바에서 검색어를 입력했을 때 결과를 보여주는 화면입니다.
포함될 UI/기능:
검색어(교사, 교실, 해시태그)에 따른 DB 조회 결과 리스트 출력

5. MyPage.jsx (마이페이지)

흐름도 위치: 우측 하단 마이페이지 구역
역할: 개인화된 정보를 관리하는 화면입니다. 내용이 많아지면 폴더로 묶어서 하위 페이지로 나눌 수도 있습니다.
포함될 탭 또는 하위 페이지:
내 시간표 (입력 및 수정)
계획서 (작성)
알람 (시간 설정)

6. AdminPage.jsx (관리자 페이지)

흐름도 위치: 좌측 상단 ADMIN 구역
역할: ADMIN 계정으로 로그인했을 때만 접근 가능한 페이지입니다.
포함될 UI/기능:
시간표 데이터 수정 폼
유저 정보 관리 테이블
